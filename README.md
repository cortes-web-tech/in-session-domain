# inSession

## An open source presentation management solution [Demo](https://inSessionEvent.com)

## Features

- _Displaying event data_

## Tech Stack

- Built on LAMP
- ReactJS Front end
- Tailwind CSS
- Hosted on AWS EC2
- Vite to reduce React bloatware
- Wails Mac App
- Customizable databases using MariaDB & MySQL
